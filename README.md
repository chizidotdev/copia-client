### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:chizidotdev/copia-client.git
   ```
   OR
   ```sh
   git clone https://github.com/chizidotdev/copia-client.git
   ```
1. Install NPM packages
   ```sh
   npm install
   ```
